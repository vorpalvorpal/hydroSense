# leachatetools: Water Quality Analysis Tools for Leachate-Impacted Waters

Tools for analysing water quality in systems potentially impacted by
landfill leachate. Core capabilities: species sensitivity distribution
(SSD) based potentially affected fraction (PAF) calculations using
ANZECC/ANZG toxicity data; multi-substance PAF (msPAF / AmsPAF) with
local background adjustment via the Added Risk Approach; and end-member
mixing analysis for leachate detection (LMF).
