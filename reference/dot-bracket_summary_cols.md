# Column names of the bracket envelope summary for a given mode

Single source of truth for the per-mode summary schema, shared by
[`.summarise_bracket()`](https://vorpalvorpal.github.io/leachatetools/reference/dot-summarise_bracket.md)'s
consumers and
[`.empty_daily_result()`](https://vorpalvorpal.github.io/leachatetools/reference/dot-empty_daily_result.md).

## Usage

``` r
.bracket_summary_cols(gap_uncertainty = "bracket")
```
