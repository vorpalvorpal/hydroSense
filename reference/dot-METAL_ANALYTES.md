# All metal-type analytes (the metals group in [`leachate_impute_groups()`](https://vorpalvorpal.github.io/leachatetools/reference/leachate_impute_groups.md))

A leachate-preset default: these define the `metals`
[`impute_group()`](https://vorpalvorpal.github.io/leachatetools/reference/impute_group.md)'s
targets and presence hurdle. For other domains, build your own groups
with
[`impute_group()`](https://vorpalvorpal.github.io/leachatetools/reference/impute_group.md)
instead.

## Usage

``` r
.METAL_ANALYTES
```

## Format

An object of class `character` of length 22.
